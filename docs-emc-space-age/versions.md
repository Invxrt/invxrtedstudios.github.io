## Update Policy

- **Major Updates**: New planets, significant features, major mod additions
- **Minor Updates**: Balance changes, quest additions, mod updates
- **Hotfixes**: Critical bug fixes and space-specific issues
- **Space Patches**: Dimension loading and performance optimizations

## Changelog Format

Each version will include:
- **Added**: New features, mods, and planets
- **Changed**: Modifications to existing systems
- **Fixed**: Bug fixes and optimizations
- **Removed**: Deprecated features or mods
- **Balanced**: EMC value adjustments and recipe changes
