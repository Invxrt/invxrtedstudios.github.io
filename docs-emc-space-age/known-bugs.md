## Space-Specific Issues to Watch

When exploring space, be aware of potential issues:
- **Dimension Loading**: Traveling between planets may cause temporary lag
- **Oxygen Systems**: Complex oxygen setups can sometimes have edge cases
- **Rocket Rendering**: Some visual glitches may occur with rockets
- **Gravity Changes**: Rapid gravity changes can cause momentary client desyncs

Report any consistent issues you encounter!

## Reporting Bugs

Found a bug? Here's how to report it:

1. **Check This Page**: Make sure it's not already listed
2. **CurseForge**: Report on the [modpack page](https://www.curseforge.com/minecraft/modpacks/emc-space-age)
3. **Discord**: Join the Discord server for quick support
4. **Include Details**:
   - Which planet were you on?
   - What were you doing when the bug occurred?
   - Can you reproduce it consistently?
   - Your game version and any recent changes
   - Screenshots or crash logs if applicable
   - Client or server?
