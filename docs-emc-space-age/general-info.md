# General Information

## System Requirements

### Minimum Requirements
- **Minecraft Version**: 1.20.1 (Forge)
- **RAM**: 8GB allocated
- **Java Version**: Java 17 or higher

### Recommended Requirements
- **RAM**: 10-12GB allocated
- **CPU**: Modern quad-core processor or better
- **Storage**: 6GB available space
- **GPU**: Dedicated graphics card recommended

## Installation

1. Download and install the [CurseForge App](https://www.curseforge.com/download/app)
2. Search for "EMC: Space Age" in the modpacks section
3. Click "Install" and wait for the download to complete
4. Launch the modpack from the CurseForge App
5. Allocate at least 8GB of RAM in the settings (10GB+ recommended)

## Included Mods

This modpack features 150+ mods including:

### Space & Exploration
- **Ad Astra**: Space travel and planetary exploration
- **Create**: Advanced mechanical automation
- **Mekanism**: Industrial processing for space resources
- **ProjectE**: EMC system adapted for space resources

### Technology & Automation
- **Applied Energistics 2**: Advanced storage networks
- **Refined Storage**: Alternative storage solutions
- **Thermal Series**: Energy, machines, and item transport
- **PneumaticCraft**: Pressurized systems and drones

### Utility & Quality of Life
- **JEI (Just Enough Items)**: Recipe viewing
- **FTB Quests**: Space-themed quest system
- **Waystones**: Fast travel (planet-specific)
- **WTHIT**: Block and entity information

## Space-Specific Features

### Oxygen Systems
- Oxygen is required for survival in space
- Build life support systems for bases
- Portable oxygen tanks for exploration

### Gravity
- Different planets have different gravity levels
- Some planets have no gravity (floating mechanics)
- Gravity affects movement and building

### Resources
- Unique ores and resources on each planet
- EMC values adjusted for space resources
- Rare materials only found on distant worlds

## Multiplayer

This modpack supports multiplayer! Server requirements:
- Minimum 10GB RAM allocated
- Use server files from CurseForge
- Strong CPU recommended due to multiple dimensions
- Regular backups essential

## Modpack Philosophy

EMC: Space Age focuses on:
- **Exploration**: Discover new worlds and resources
- **Technology**: Build advanced systems for space survival
- **Progression**: Carefully designed path from Earth to the stars
- **Integration**: EMC works seamlessly with space resources
