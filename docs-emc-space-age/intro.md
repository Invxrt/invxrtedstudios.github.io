# EMC: Space Age

Welcome to the **EMC: Space Age** modpack documentation!

## Overview

EMC: Space Age takes the Energy-Matter Curation concept to the stars! Explore alien worlds, build space stations, and harness the power of EMC across the cosmos in this space-themed adventure.

## Key Features

- **Space Exploration**: Travel to multiple planets and dimensions
- **EMC Across the Stars**: Use ProjectE's EMC system enhanced for space travel
- **Advanced Technology**: Access to high-tech mods for space colonization
- **Planetary Bases**: Build bases on different planets with unique challenges
- **Galactic Quests**: Follow a cosmic storyline through the quest system
