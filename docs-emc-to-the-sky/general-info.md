# General Information

## System Requirements

### Minimum Requirements
- **Minecraft Version**: 1.20.1 (Forge)
- **RAM**: 6GB allocated
- **Java Version**: Java 17 or higher

### Recommended Requirements
- **RAM**: 8-10GB allocated
- **CPU**: Modern quad-core processor
- **Storage**: 5GB available space

## Installation

1. Download and install the [CurseForge App](https://www.curseforge.com/download/app)
2. Search for "EMC: To The Sky!" in the modpacks section
3. Click "Install" and wait for the download to complete
4. Launch the modpack from the CurseForge App
5. Allocate at least 6GB of RAM in the settings

## Included Mods

This modpack features over 100+ mods including:

### Core Mods
- **ProjectE**: The foundation of the EMC system
- **Applied Energistics 2**: Advanced storage and automation
- **Mekanism**: Industrial processing and power generation
- **Thermal Expansion**: Machines and item transport

### Utility Mods
- **JEI (Just Enough Items)**: Recipe viewing
- **FTB Quests**: Quest system and progression tracking
- **Waystones**: Fast travel between locations
- **WTHIT**: Block and entity information

## Multiplayer

This modpack is fully compatible with multiplayer servers. Server owners should:
- Allocate at least 8GB of RAM
- Use the server files provided on CurseForge
- Keep backups regularly due to skyblock nature

## Modpack Philosophy

EMC: To The Sky! is designed to provide a challenging but fair skyblock experience. The pack focuses on:
- **Progression**: Carefully gated progression through quests
- **Balance**: EMC values are adjusted to prevent early-game exploits
- **Variety**: Multiple paths to achieve your goals
- **Automation**: Encourage building automated systems
