# EMC: To The Sky!

Welcome to the **EMC: To The Sky!** modpack documentation!

## Overview

EMC: To The Sky! is a skyblock adventure modpack that combines the classic skyblock experience with Energy-Matter Curation (EMC) systems. Build your island empire from nothing using EMC, automation, and expert-mode crafting challenges.

## Key Features

- **Skyblock Progression**: Start with minimal resources and expand your island
- **EMC System**: Use ProjectE's Energy-Matter Curation to transmute items
- **Expert Mode Crafting**: Carefully balanced recipe changes for a challenging experience
- **Automation Focus**: Utilize tech mods to automate your resource generation
- **Quest System**: Follow guided quests to progress through the pack
