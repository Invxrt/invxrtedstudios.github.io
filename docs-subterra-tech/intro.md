# SubTerra Tech

Welcome to the **SubTerra Tech** modpack documentation!

## Overview

SubTerra Tech is an underground-focused tech modpack where you'll delve deep beneath the surface to discover resources, build advanced technology, and create massive subterranean bases. The deeper you go, the more powerful the technology you can unlock!

## Key Features

- **Underground Progression**: Start on the surface, progress deeper underground
- **Depth-Based Technology**: Unlock better tech as you mine deeper
- **Massive Cave Systems**: Explore vast underground networks
- **Industrial Automation**: Build complex automated mining and processing systems
- **Cave Base Building**: Create impressive underground facilities
