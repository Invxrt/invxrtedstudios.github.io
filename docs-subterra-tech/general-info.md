# General Information

## System Requirements

### Minimum Requirements
- **Minecraft Version**: 1.20.1 (Forge)
- **RAM**: 6GB allocated
- **Java Version**: Java 17 or higher

### Recommended Requirements
- **RAM**: 8GB allocated
- **CPU**: Modern quad-core processor
- **Storage**: 5GB available space

## Installation

1. Download and install the [CurseForge App](https://www.curseforge.com/download/app)
2. Search for "SubTerra Tech" in the modpacks section
3. Click "Install" and wait for the download to complete
4. Launch the modpack from the CurseForge App
5. Allocate at least 6GB of RAM in the settings

## Depth Mechanics

SubTerra Tech introduces unique depth-based progression:

### Surface Layer (Y: 64 and above)
- Basic resources and starting technology
- Standard overworld ores
- Quest beginning point
- Surface base establishment

### Shallow Underground (Y: 32 to 64)
- Enhanced ore generation
- First tier of advanced machinery unlocks
- Cave systems begin
- Light mining automation available

### Deep Underground (Y: 0 to 32)
- Rare ore deposits
- Second tier technology access
- Larger cave networks
- Advanced automation possible

### Deepslate Layer (Y: -16 to 0)
- Valuable resources
- Third tier machinery unlocks
- Complex cave formations
- Requires advanced gear

### Deep Dark (Y: -64 to -16)
- Rarest resources
- End-game technology
- Ancient City integration
- Highest-tier automation
- Requires top-tier equipment

## Multiplayer

This modpack is multiplayer-friendly! Server requirements:
- Minimum 8GB RAM allocated
- Use server files from CurseForge
- Regular backups recommended
- Consider pre-generating chunks for better performance

## Modpack Philosophy

SubTerra Tech focuses on:
- **Vertical Progression**: Technology unlocks as you go deeper
- **Industrial Automation**: Complex automated systems
- **Underground Building**: Creating impressive subterranean bases
- **Resource Management**: Efficient mining and processing
- **Exploration**: Discovering the depths and their secrets
