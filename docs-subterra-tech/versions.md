# Version History

This page tracks all versions and updates for SubTerra Tech.

## Version 1.0.0

**Release Date**: TBD

### Initial Release
- Initial modpack release
- 120+ mods included
- Depth-based progression system
- Complete quest line from surface to deep dark
- Enhanced cave generation
- Industrial automation focus

### Cave Systems
- YUNG's Better Caves integration
- Deeper and Darker content
- Ancient City enhancements
- Multi-level cave networks

### Technology Features
- Mekanism 5x ore processing
- Create mechanical automation
- Thermal Expansion machinery
- Industrial Foregoing systems
- Applied Energistics 2 storage

### Progression System
- Surface layer (Y: 64+)
- Shallow underground (Y: 32-64)
- Deep underground (Y: 0-32)
- Deepslate layer (Y: -16-0)
- Deep dark (Y: -64--16)

## Update Policy

- **Major Updates**: New depth mechanics, major features, significant mod additions
- **Minor Updates**: Balance adjustments, quest additions, mod updates
- **Hotfixes**: Critical bug fixes and performance issues
- **Cave Patches**: Generation and performance optimizations

## Changelog Format

Each version will include:
- **Added**: New features, mods, and mechanics
- **Changed**: Modifications to existing systems
- **Fixed**: Bug fixes and optimizations
- **Removed**: Deprecated features or mods
- **Balanced**: Recipe changes and depth-based adjustments
